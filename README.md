- Passport

- Mongoose.js
- MongoDB
- Gulp
- Browserify
- Babel



### Usage Instructions

#### To run the app,
1. Make sure `mongodb` is running in the background.
2. Install global dependencies with `npm install -g gulp react-tools browserify`
3. Install dependencies with `npm install`
4. Run app with `gulp serve`
